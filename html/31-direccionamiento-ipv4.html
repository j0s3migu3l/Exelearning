3.1. Direccionamiento IPv4

1. Características

  • 32 bits (4 números entre 0–255)
  • Ejemplo: 192.168.1.1


 🧮 Clases de IP

  • Clase A → redes grandes
  • Clase B → redes medianas
  • Clase C → redes pequeñas
  • Clase D → multicast
  • Clase E → experimental


 🔒 Direcciones privadas

  • 10.0.0.0 – 10.255.255.255
  • 172.16.0.0 – 172.31.255.255
  • 192.168.0.0 – 192.168.255.255

2. Máscara de red

Sirve para distinguir:

  • Parte de red
  • Parte de host


Ejemplo:

IP: 192.168.1.10
Máscara: 255.255.255.0
👉 Permite saber qué equipos están en la misma red

📌 Notación CIDR

Ejemplo:

192.168.1.10/24 → máscara 255.255.255.0

 
🧠 Cálculo de red (AND lógico)

Para comprobar si dos equipos están en la misma red:

  • Se hace AND entre IP y máscara
  • Si el resultado es igual → misma red

 
📢 Direcciones especiales

Dirección de red → host = 0
Broadcast → host = 1 (todos)
👉 No se pueden asignar a equipos

Ejercicios a realizar:

🧠 Ejercicio 3: Calcular red con máscara distinta
IP: 192.168.1.130/25

 
🧠 Ejercicio 4: Número de hosts en una red
Red: 10.0.0.0/16

Ejercicio 1

Dada la IP: 192.168.1.25/24

👉 Paso 1: Máscara
/24 → 255.255.255.0

👉 Paso 2: Dirección de red
➡ 192.168.1.0

👉 Paso 3: Broadcast
➡ 192.168.1.255

👉 Paso 4: Hosts válidos
➡ Desde 192.168.1.1 hasta 192.168.1.254

👉 Paso 5: Número de hosts
➡ 2^8−2=254

Ejercicio 2

👉 Solución
Ambas tienen:

Red → 192.168.1.0
✔ Sí, están en la misma red

Ejercicio 3

192.168.1.130/25

👉 Paso 1: Máscara
/25 → 255.255.255.128

👉 Paso 2: Subredes posibles
192.168.1.0 – 127
192.168.1.128 – 255

👉 Paso 3: Identificar red
130 pertenece al segundo rango:

➡ Red: 192.168.1.128
➡ Broadcast: 192.168.1.255

👉 Hosts
➡ Desde 192.168.1.129 hasta 192.168.1.254
