3. Direccionamiento IP

1. ¿Qué es una dirección IP?
Una dirección IP es un número único que identifica a cada equipo dentro de una red.

Formato: 32 bits (IPv4)
Se representa en 4 números decimales (0–255) separados por puntos


👉 Ejemplo: 192.168.1.1
 
2. Estructura de una IP
Una dirección IP se divide en dos partes:

Parte de red → identifica la red
Parte de host → identifica el equipo dentro de esa red
👉 Esta división depende de la máscara de red

 
3. Clases de direcciones IP
🔸 Clase A
Redes muy grandes
Primer bit = 0
Rango: 1 – 126
Muchos hosts por red
🔸 Clase B
Redes medianas
Primeros bits = 10
Rango: 128 – 191
🔸 Clase C
Redes pequeñas
Primeros bits = 110
Rango: 192 – 223
🔸 Clase D
Multidifusión (multicast)
🔸 Clase E
Uso experimental
 
4. Direcciones IP privadas
Se usan en redes internas (no en Internet):

10.0.0.0 – 10.255.255.255
172.16.0.0 – 172.31.255.255
192.168.0.0 – 192.168.255.255
 
5. Máscara de red
La máscara de red indica qué parte de la IP es red y cuál es host.

👉 Ejemplo:

IP: 192.168.1.10
Máscara: 255.255.255.0
✔ Los bits en 1 → red
✔ Los bits en 0 → host

 
6. Notación CIDR
Forma abreviada de escribir IP + máscara:

👉 Ejemplo:

192.168.1.10/24 = máscara 255.255.255.0
✔ El número indica los bits en 1 de la máscara
