3.2. Direccionamiento IPv6

1. ¿Qué es IPv6

IPv6 es la evolución de IPv4 y se creó para solucionar el problema de escasez de direcciones IP.

Tamaño: 128 bits (mucho mayor que IPv4)
Permite ≈ 3.4 × 10³⁸ direcciones
Mejora seguridad, eficiencia y autoconfiguración

 
2. Formato de una dirección IPv6

Se representa en hexadecimal (base 16):

👉 Ejemplo:

 
2001:0db8:85a3:0000:0000:8a2e:0370:7334
 
8 bloques de 16 bits
Separados por “:”
Cada bloque → 4 dígitos hexadecimales

 
3. Simplificación de direcciones

IPv6 permite abreviar:

✔ Quitar ceros a la izquierda
 
2001:0db8 → 2001:db8
 
✔ Sustituir bloques de ceros por "::"
 
2001:db8:0:0:0:0:2:1 → 2001:db8::2:1
 
⚠ Solo se puede usar una vez "::" por dirección

 
4. Estructura de IPv6

Se divide en dos partes principales:

Prefijo de red (identifica la red)
Identificador de interfaz (equipo)


👉 Ejemplo:

2001:db8::/64
 
/64 → primeros 64 bits = red
últimos 64 bits = host

 
5. Tipos de direcciones IPv6

🔸 Unicast (uno a uno)
Identifica un único dispositivo

Global (Internet)
Link-local (solo red local)
🔸 Multicast (uno a varios)
Envía datos a un grupo de equipos

🔸 Anycast
Envía datos al equipo más cercano de un grupo

❌ No existe broadcast en IPv6 (se sustituye por multicast)

 
6. Direcciones importantes

✔ Loopback
 
::1
 
→ equivalente a 127.0.0.1 en IPv4

✔ Link-local
 
fe80::/10
 
→ comunicación dentro de la red local

✔ Global unicast
 
2000::/3
 
→ direcciones públicas en Internet

2. Ejercicios de IPv6

Ejercicios de IPv6:

🧠 Ejercicio 1: Simplificar una dirección IPv6
Dirección:

 2001:0db8:0000:0000:0000:ff00:0042:8329
 
 🧠 Ejercicio 2: Expandir una dirección IPv6
Dirección:

2001:db8::1

🧠 Ejercicio 3: Identificar tipo de dirección
Dirección:

 fe80::1a2b:3c4d
 
🧠 Ejercicio 4: Identificar red e interfaz
Dirección:

 2001:db8:abcd:0012:0000:0000:0000:0001/64
 

Ejercicio 1

👉 Paso 1: Quitar ceros a la izquierda
 
2001:db8:0:0:0:ff00:42:8329
 
👉 Paso 2: Sustituir bloques de ceros consecutivos
 
2001:db8::ff00:42:8329
 
✔ Resultado final:
➡ 2001:db8::ff00:42:8329

Ejercicio 2

👉 Paso 1: Contar bloques
IPv6 tiene 8 bloques
Actualmente hay 2 bloques + "::" + 1 bloque = faltan 5 bloques
👉 Paso 2: Añadir ceros
 
2001:0db8:0000:0000:0000:0000:0000:0001

Ejercicio 3

👉 Solución
Empieza por fe80
Pertenece al rango fe80::/10
✔ Tipo: Link-local
➡ Solo válida dentro de la red local

Ejercicio 4

👉 Paso 1: Simplificar
 
2001:db8:abcd:12::1
 
👉 Paso 2: Separar partes
Prefijo /64 → primeros 64 bits (4 bloques)
✔ Red:

 
2001:db8:abcd:12::
 
✔ Identificador de interfaz:

 
::1
